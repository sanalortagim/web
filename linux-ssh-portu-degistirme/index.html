    Linux SSH Portu Değiştirme

    Linux SSH Varsayılan Portu Değiştirme

    Uyarı : Yeni belirlemiş olduğunuz ssh portunuzun bilinmiş veya engellenmiş portlar ile çakışmaması için kontrollerinizi gerçekleştiriniz.

    İlk olarak ssh bağlantınızı gerçekleştirin ve root olarak giriş yapın.

    Sonrasında şu komutu uygulayalım.

    vi /etc/ssh/sshd_config

    Şu satırı bulun.

    # Port 22

    Satırda bulunan # işaretini kaldırıp ve portumuzu belirliyoruz. Örnek aşağıda gösterilmiştir.

    Port 40

    Bu değişikliklerin aktif olabilmesi için ssh servisini yeniden başlatmamız gerekiyor.Ardından yeni portumuz ile bağlantı kurabiliriz.

    service sshd restart
