T-SQL Sayının Tek veya Çift Olduğunu Bulan Örnek

30 Kasım 2019 tarihinde yazıldı. ( 1 dakika )


T-SQL ‘de değişkene atadığımız sayının tek veya çift olduğunu bulan program aşağıdaki gibidir.

T-SQL Kodu :

Declare @sayi INT
Set @sayi = 52

If @sayi % 2 = 0
	Print 'Sayı çifttir.'
Else
	Print 'Sayı tektir'
Go
